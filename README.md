## Explication des principes SOLID

- **SRP :** Une classe doit avoir une seule responsabilité.
- **OCP :** Le code doit être extensible sans être modifié.
- **LSP :** Les enfants doivent pouvoir remplacer les parents.
- **ISP :** Préférer plusieurs petites interfaces à une seule grosse.
- **DIP :** Dépendre des abstractions, pas des implémentations.
